Introduction to Html

HTTML stands for HyperText Markup Language. It is the standard language used to create web pages and tell web browsers how to display content (text, images, links, videos, etc.).
What Does HTML Do?
Structures the content on the web.
Organizes text, images, links, forms, etc.
Works with CSS (for design) and JavaScript (for interactivity)
